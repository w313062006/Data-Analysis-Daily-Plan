# Data-Analysis-Daily-Plan
# 吾圆数据分析师培养计划

## What is it?
这是一份学习数据分析的计划书，通过这份计划可以学习到基本的：
- 商务统计的思想
- 数据挖掘基本流程
- 使用Python/R进行数据分析
- Sql（可选）

**这份计划为期12周。**
## How to use it?
这是一份从上到下的计划，每行代表一个里程碑，然后进行下一里程碑的学习。
